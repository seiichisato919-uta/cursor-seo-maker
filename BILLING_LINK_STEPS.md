# 請求アカウントをリンクする手順（現在の画面から）

## 現在の画面から次のステップ

### ステップ1: 左側のサイドバーから「設定」をクリック

1. 画面左側のサイドバーを下にスクロール
2. 「設定」という項目を探す（歯車のアイコンが付いている可能性があります）
3. 「設定」をクリック

### ステップ2: 請求セクションを探す

1. 設定ページが開いたら、「請求」または「Billing」というセクションを探す
2. 「請求アカウントをリンク」または「請求アカウントを変更」というボタンがあるはずです

### ステップ3: 請求アカウントを選択

1. 「請求アカウントをリンク」をクリック
2. 「請求先アカウント2」を選択
3. 「設定」または「リンク」をクリック

## 別の方法：直接URLにアクセス

もし「設定」が見つからない場合は、以下のURLに直接アクセスしてください：

```
https://console.cloud.google.com/billing/projects
```

このページで：
1. 「my-seo-system」プロジェクトを探す
2. 「請求アカウント」列を確認
3. 「請求アカウントなし」と表示されている場合は、その部分をクリック
4. 「請求先アカウント2」を選択
5. 「設定」をクリック

## 確認方法

請求アカウントがリンクされたら：
1. 設定ページの「請求」セクションで「請求先アカウント2」が表示されることを確認
2. Generative Language APIが有効になっていることを確認（現在の画面で「ステータス: 有効」になっていることを確認）
