# APIキーの割り当てティアを有料プランに変更する方法

## 問題の原因

APIキーの「割り当てティア」が「無料枠」になっているため、有料プランにアップグレードしても無料枠の制限が適用されています。

## 解決方法

### 方法1: 既存のAPIキーの割り当てティアを変更（推奨）

1. Google AI StudioのAPIキーページで、「my-seo-system」のAPIキーを探す
2. 「割り当てティア」列の「お支払い情報を設定」をクリック
3. 有料プランに変更する手順に従う
4. 変更が反映されるまで数分待つ

### 方法2: 新しいAPIキーを作成

1. Google AI StudioのAPIキーページで「APIキーを作成」をクリック
2. プロジェクトを選択（「my-seo-system」を選択）
3. 新しいAPIキーをコピー
4. `.env.local`ファイルを開く
5. `GEMINI_API_KEY`の値を新しいAPIキーに更新
6. ファイルを保存
7. 開発サーバーを再起動

## 確認方法

APIキーの「割り当てティア」が以下のいずれかになっていることを確認：
- 「Tier 1」（有料プラン）
- 「お支払い情報を設定」が表示されていない（有料プランが有効）

## 次のステップ

1. APIキーの割り当てティアを変更、または新しいAPIキーを作成
2. `.env.local`ファイルを更新（新しいAPIキーを作成した場合）
3. 開発サーバーを再起動
4. 再度試す
